How to Make a Heyzap iOS SDK Plugin

For ultimate flexibility, we have made the Heyzap SDK extensible so that you can customize its behavior to your needs. All you need to do is subclass the class HeyzapSDK:

@interface HeyzapCustomSDK : HeyzapSDK <HeyzapDelegate>

This subclass should conform to the HeyzapDelegate protocol. Then you can implement the following methods:

- (void) heyzapWillAppear: (BOOL) animated;
- (void) heyzapDidAppear: (BOOL) animated;
- (void) heyzapWillDisappear: (BOOL) animated;
- (void) heyzapDidDisappear: (BOOL) animated;

All the methods are optional and are called if they exist inside your subclass, and behave in the same way that UIView works. From there, everything will work just as the normal HeyzapSDK does.
